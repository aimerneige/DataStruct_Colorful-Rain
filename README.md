# DataStruct_Colorful-Rain
此代码库用于数据结构课设--池塘夜降彩色雨 项目的交流和代码共建
要求：
  1.成员提交每次课设实验后的  经验 和 心得   到  issue里
  2.成员以issue形式提交每次实验后的可用代码段（结构体以及其对应的操作（函数）），由xikebyme整合成型后放入代码库，最后大家可下载到自己电脑进行整体调试和相应改进，形成各自不同的成品。。。
